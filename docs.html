快速开始

欢迎使用 LMS 平台！本指南将帮助您快速上手并开始创建您的第一个在线课程。

开始之前

确保您已拥有 LMS 账户。如果没有，请前往注册页面创建一个免费账户。

第一步：创建账户

访问注册页面，填写必要信息后，您将收到一封确认邮件。点击邮件中的链接即可激活您的账户。

第二步：登录平台

使用您的邮箱和密码登录 LMS 平台。首次登录后，系统将引导您完成基本设置。

第三步：创建课程

在控制台中点击"新建课程"，按照提示填写课程信息，上传课程内容，即可开始教学！

安装指南

本节介绍如何在不同环境中安装和配置 LMS 平台。

系统要求

  • Node.js 18.0 或更高版本
  • npm 9.0 或更高版本
  • 现代浏览器（Chrome、Firefox、Safari、Edge）

通过 npm 安装

Terminal 
$ npm install lms-platform

通过 Yarn 安装

Terminal 
$ yarn add lms-platform

初始化项目

Terminal 
$ npx lms-platform init
$ npm run dev

配置说明

了解如何配置 LMS 平台以满足您的需求。

基础配置

config.json 
{
  "appName": "My LMS",
  "language": "zh-CN",
  "timezone": "Asia/Shanghai",
  "theme": {
    "primaryColor": "#3B82F6",
    "darkMode": true
  }
}

高级配置

您可以通过环境变量覆盖配置文件中的设置：

.env 
LMS_API_URL=https://api.lms-platform.com
LMS_API_KEY=your_api_key_here
LMS_DEBUG=false

课程创建

学习如何在 LMS 平台上创建和管理在线课程。

创建新课程

使用以下 API 端点创建新课程：

HTTP Request 
POST /api/v1/courses
Content-Type: application/json
Authorization: Bearer {token}

{
  "title": "Web 开发实战",
  "description": "从零开始学习 Web 开发",
  "category": "前端",
  "price": 199
}

添加课程章节

HTTP Request 
POST /api/v1/courses/{courseId}/chapters
Content-Type: application/json
Authorization: Bearer {token}

{
  "title": "第一章：HTML 基础",
  "order": 1,
  "content": "..."
}

API 参考

完整的 RESTful API 文档，帮助您集成 LMS 平台到您的应用程序。

GET /api/v1/courses

获取课程列表

查询参数：

参数 类型 必填 说明
page integer 否 页码，默认 1
limit integer 否 每页数量，默认 10
POST /api/v1/courses

创建新课程

请求体：

{
  "title": "课程标题",
  "description": "课程描述",
  "category": "课程分类"
}
PUT /api/v1/courses/{id}

更新课程信息

DELETE /api/v1/courses/{id}

删除课程
